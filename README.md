# ConfereSmart — PWA

Sistema de conferência de pedidos DAV com scanner de código de barras.

## Como hospedar (GitHub Pages — gratuito)

### 1. Criar conta no GitHub
- Acesse https://github.com e crie uma conta gratuita

### 2. Criar repositório
- Clique em **"New repository"**
- Nome: `confresmart` (ou qualquer nome)
- Marque **"Public"**
- Clique em **"Create repository"**

### 3. Fazer upload dos arquivos
- Na página do repositório, clique em **"uploading an existing file"**
- Arraste todos estes arquivos:
  - `index.html`
  - `manifest.json`
  - `sw.js`
  - `icon-192.png`
  - `icon-512.png`
- Clique em **"Commit changes"**

### 4. Ativar GitHub Pages
- Vá em **Settings** → **Pages**
- Em "Source", selecione **"Deploy from a branch"**
- Branch: **main** / pasta: **/ (root)**
- Clique em **Save**

### 5. Aguardar (1-2 minutos)
- A URL do app será: `https://SEU-USUARIO.github.io/confresmart/`

---

## Instalar no celular

1. Abra a URL acima no **Chrome Android**
2. Toque nos **3 pontinhos** (⋮)
3. Toque em **"Adicionar à tela inicial"**
4. Confirme — o ícone dourado aparece na tela!

## Funcionalidades
- 📋 Conferência de pedidos DAV em PDF
- 📷 Scanner de código de barras pela câmera
- ⌨️ Digitação manual de código
- 📊 Relatório com taxa de acerto
- 📲 Envio por WhatsApp
- 🕘 Histórico de conferências
- 📱 Funciona offline após primeiro acesso
